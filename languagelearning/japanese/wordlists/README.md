# 日语词库

本目录用于存放日语学习词库文件。

## 词库格式要求

每行一个单词，格式为：
```
単語 [たんご] 单词、词汇
```

- 日语汉字/假名
- [读音]（假名或罗马音）
- 中文释义

## 示例

```
こんにちは [konnichiha] 你好
ありがとう [arigatou] 谢谢
学生 [がくせい] 学生
先生 [せんせい] 老师
```

## 词库文件命名建议

- N5.txt - JLPT N5词汇
- N4.txt - JLPT N4词汇
- N3.txt - JLPT N3词汇
- N2.txt - JLPT N2词汇
- N1.txt - JLPT N1词汇

## 添加词库

1. 将词库文件放入本目录
2. 修改 `../japanese.js` 中的 `VOCABULARY_BOOKS` 数组
3. 添加对应的词库配置信息

## 注意事项

- 文件编码必须为UTF-8
- 每行格式必须严格遵守
- 读音部分可以是假名或罗马音
