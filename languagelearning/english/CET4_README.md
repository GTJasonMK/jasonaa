# CET4词汇书AI增强版使用指南

## 概述

CET4词汇书AI增强版包含4544个大学英语四级核心词汇，每个单词都配备了AI生成的深度学习内容。

## 功能特点

### 1. 基础功能
- **随机抽词练习**：从词库中随机抽取单词进行记忆测试
- **认识/不认识判断**：快速标记单词掌握情况
- **学习进度跟踪**：自动记录已掌握和未掌握的单词数量
- **历史导航**：前后翻页查看最近练习的50个单词

### 2. AI增强功能（每个单词）

#### 相近释义及区别
点击"相近释义及区别"按钮，AI会为你分析：
- 该单词的主要同义词（3-5个）
- 每个同义词的音标、核心区别和使用场景
- 对比例句展示细微差异
- 使用建议和记忆技巧

**示例：** 单词"abandon"
```
同义词对比：
- desert: 强调违背责任，常用于人际关系
- forsake: 文学性强，多用于正式或宗教语境
- discard: 强调因无用而丢弃，多用���物品
- relinquish: 正式用语，常用于法律或权利转让
```

#### 短语及用法
点击"短语及用法"按钮，AI会提供：
- 6-8个常用短语搭配
- 每个短语的音标、中文翻译和例句
- 使用场景说明（口语/书面语，正式/非正式）
- 固定搭配的介词或助词

**示例：** 单词"abandon"
```
常用短语：
1. abandon oneself to - 沉溺于
2. abandon ship - 弃船；放弃计划
3. abandon hope - 放弃希望
4. abandon the idea - 打消念头
...
```

#### 自定义问题
在"问其他问题"输入框中，你可以问任何关于当前单词的问题：
- "这个单词有哪些常见的词根词缀？"
- "能给我几个高考真题例句吗？"
- "这个词在美式英语和英式英语中有区别吗？"
- "如何区分这个词的名词和动词用法？"

### 3. 响应历史导航
- AI的回复会保存在历史记录中
- 使用「⬅️」和「➡️」按钮在历史回复之间切换
- 当前查看的是第几条回复会显示在底部

## 使用方法

### 启动练习

1. **访问英语学习页面**
   ```
   http://localhost:8080/languagelearning/english/index.html
   ```

2. **选择词汇书**
   - 在词汇书列表中选择"大学英语四级（AI增强版）"
   - 点击"开始练习"

3. **开始学习**
   - 查看单词、音标和释义
   - 判断是否认识该单词
   - 点击"认识"或"不认识"进入下一个单词

### 使用AI助手

1. **查询同义词辨析**
   - 点击右上角AI面板中的"相近释义及区别"按钮
   - 等待1-3秒，AI回复会显示在页面底部
   - 仔细阅读同义词对比表和使用建议

2. **查询短语搭配**
   - 点击"短语及用法"按钮
   - 查看常用短语、例句和使用场景
   - 特别注意固定搭配的介词

3. **自定义查询**
   - 在"问其他问题"输入框中输入你的问题
   - 按Enter或点击输入框外区域发送
   - AI会针对当前单词回答你的问题

4. **查看历史回复**
   - 点击「⬅️」查看上一条回复
   - 点击「➡️」查看下一条回复
   - 底部显示当前查看的回复编号

### 学习进度管理

**进度统计**（显示在页面顶部）
```
已练习: 50/4544 词
认识: 30  不认识: 20
```

**重置进度**
- 点击"退出练习"返回词汇书选择页面
- 重新进入会保留之前的进度
- 如需重置，请在浏览器开发者工具中清除LocalStorage

## 技术细节

### 数据结构

**分片存储**
- CET4词汇被分成10个分片，每片约450词
- 每个分片文件约3.1MB，符合GitHub 25MB限制
- 前端自动并发加载所有分片并合并

**数据格式**
```json
{
  "word": {
    "synonyms": {
      "content": "Markdown格式的同义词分析",
      "generated": "2025-11-05T12:00:00.000Z"
    },
    "phrases": {
      "content": "Markdown格式的短语搭配",
      "generated": "2025-11-05T12:01:00.000Z"
    }
  }
}
```

### AI配置

AI功能需要配置API服务：

1. **打开AI聊天工具**
   ```
   http://localhost:8080/aitools/aichat/index.html
   ```

2. **配置API**
   - 点击右上角齿轮图标
   - 选择服务商（OpenAI、DeepSeek、自定义API等）
   - 填入API Key和API URL
   - 保存配置

3. **支持的服务商**
   - OpenAI (gpt-3.5-turbo, gpt-4, gpt-4o等)
   - DeepSeek (deepseek-chat)
   - 自定义OpenAI兼容API（Ollama、LM Studio等）

### 缓存机制

- AI查询结果会缓存在内存中（最多50条）
- 相同查询会直接返回缓存结果，无需重复请求
- 缓存在刷新页面后清空

## 常见问题

### Q: AI查询很慢或失败？
**A:** 可能原因：
1. 网络连接问题：检查网络是否正常
2. API限流：降低查询频率，等待片刻
3. API Key无效：重新配置正确的API Key
4. 余额不足：检查API账户余额

### Q: 为什么有些单词没有AI内容？
**A:** 约有13个单词（0.3%）在生成过程中失败，这些单词仍可正常练习，只是缺少AI增强内容。

### Q: 如何离线使用？
**A:**
1. 基础练习功能可完全离线使用
2. AI功能需要联网，但可以配置本地LLM（如Ollama）
3. 配置本地LLM后，AI功能也可离线使用

### Q: 进度数据存储在哪里？
**A:**
- 进度数据保存在浏览器的LocalStorage中
- 键名格式：`english_practice_cet4_edited_progress`
- 更换浏览器或清除浏览器数据会丢失进度

### Q: 可以导出学习记录吗？
**A:**
当前版本暂不支持导出功能，但你可以：
1. 打开浏览器开发者工具（F12）
2. 进入Console标签
3. 输入：`localStorage.getItem('english_practice_cet4_edited_progress')`
4. 复制输出的JSON数据��存

## 学习建议

### 高效学习流程

1. **第一轮：快速筛选**
   - 快速判断认识/不认识，不要犹豫
   - 目标：找出真正不认识的单词
   - 时间：10-15分钟/100词

2. **第二轮：精读不认识的词**
   - 重点学习标记为"不认识"的单词
   - 使用AI查询同义词和短语
   - 做笔记或截图保存重点内容
   - 时间：3-5分钟/词

3. **第三轮：复习巩固**
   - 重新测试之前不认识的单词
   - 查看进度统计，确保掌握率提升
   - 使用"前/后"按钮复习历史单词

### AI功能最佳实践

1. **同义词学习**
   - 重点关注"核心区别"和"使用场景"
   - 对比例句，体会细微差异
   - 尝试自己造句运用

2. **短语搭配**
   - 重点记忆高频短语（标注为常用的）
   - 注意固定搭配的介词
   - 区分口语和书面语场景

3. **自定义查询**
   - 针对自己的薄弱点提问
   - 结合真题或作文需求查询
   - 可以要求AI出练习题

### 记忆技巧

1. **词根词缀法**：问AI词源和构词规则
2. **联想记忆法**：问AI相关词汇和主题词群
3. **场景记忆法**：问AI在特定场景下的用法
4. **对比记忆法**：查询同义词辨析，避免混淆
